# DSCVR SDK

The DSCVR SDK is a TypeScript library designed to facilitate interactions with the DSCVR platform, leveraging DSCVR's Social Fabric Protocol

## Features

- Users
- Content
- Portals

## Installation


```bash
pnpm install @dscvr/dscvr-sdk
```
or
```bash
npm install @dscvr/dscvr-sdk
```
